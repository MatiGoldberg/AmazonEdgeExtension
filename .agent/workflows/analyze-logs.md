---
description: Analyze the latest logs for errors, warnings, and performance issues
---

1. Run the log analysis script to generate the report:
   ```bash
   python3 scripts/analyze_logs.py
   ```

2. Review the output report in the terminal. The report includes:
   - **Summary**: Overall health status.
   - **Critical Errors**: List of errors found.
   - **Warnings**: List of warnings.
   - **Observations**: Performance issues and patterns.
   - **Recommendations**: Suggested fixes.
